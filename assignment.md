# Docker Challenge: Deploy a Flask App
Objective:
Learn the basics of Docker by containerizing and deploying a simple Flask application from a GitHub repository.

1. Fork the Repository:
Fork the provided repository containing the Flask app to your GitHub account.
Clone the forked repository to your local machine.

2. Explore the Flask App:
Examine the Flask app code to understand its structure and functionality.
Run the Flask app locally to ensure it works. You can do this by setting up a virtual environment, installing the dependencies, and running the app.

3. Create a Dockerfile:
In the root directory of your Flask app, create a file named Dockerfile. to package flask app

4. Build the Docker image 
5. Run the Docker container from the built image, with ports, enviroment varriables and volume maps

6. create a docker hub account and push image to your private registory
